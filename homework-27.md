***Задача №13. ⭐***

Вопрос.

Опишите каждый шаг, который выполнит Angular после inject(A)
 
Начиная с поиска Provider, заканчивая кешированием.

**Ответ** Сначала идет создание класса A, либо передача уже имеющегося экземпляра. Ангуляр ищет провайдер для "А". Допустим, это первое создание экземпляра класса А. Начинается создание класса "А", но в нем есть зависимость "B". Значит, нельзя заканчивать создание класса А, так как ему нужен B. Поэтому создание A приостанавливается. Сначала нужно создать B. -> создаем экземпляр класса B, но ему нужен класс C (класс B еще не закончил создаваться) -> Не можем создать класс C, потому что ему нужен D -> начинаем создавать D, но внутри logger = inject(LoggerService); -> создаем LoggerService. Когда есть LoggerService, класс D завершает свое создание -> так как есть D, то класс C тоже завершает создание, и так выше работает с классом B и классом A.

итого получается: inject(A) -> class A not created -> class B not created -> class C not created -> class D not created -> LoggerService injected -> class D completed creation -> class C completed creation -> class B completed creation -> class A completed creation
